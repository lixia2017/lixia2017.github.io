R，Rstudio的安装与镜像设置

本文总阅读量次 2017-05-21

date: 2017-05-12 12:15:52

R和Rstudio的下载，安装比较简单，其设置相对复杂，本知识点主要讲解下其相关设置。

R下载地址: https://www.r-project.org/
Rstuido下载地址： https://www.rstudio.com/

  1. 不仅R提供了mirror下载和选择地区，Rstudio也可以选择mirror，不过其CRAN mirror默认是针对global的，需要设置为当地或者附近的。

  2. R中的bioconductor其实也提供了镜像，其方法具体如下:
    source(“http://bioconductor.org/biocLite.R“)
    options(BioC_mirror=”http://mirrors.ustc.edu.cn/bioc/“)
    biocLite(“RGalaxy”) ##这样就用中科大的镜像来下载包啦

  3. 可以再Rprofile.site文件中设置国内的镜像站点.
    Windows中该文件在D:\R\R-3.3.2\etc目录中,(这是我的安装位置,仅供参考)

set a CRAN mirror

local({r <- getOption(“repos”)

r[“CRAN”] <- “http://my.local.cran“

options(repos=r)})

别人的（供比较用）

set a CRAN mirror

local({r <- getOption(“repos”)
r[“CRAN”] <- “http://cran.csdb.cn/“
options(repos=r)}

国内站点目前有清华和中科大等3个机构的。
