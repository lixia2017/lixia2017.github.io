shell下批量处理文件

本文总阅读量次 2017-05-26

shell 批量改文件名
需要批量处理文件，记得使用while read id; do; done

##1. 在文件夹shell下创建文件10个文件
$ touch tmp_{1..10}.txt

##2. 找到含有“_“”的文件夹。

$ find ./ -name “_“

##3. 读取id, 用mv改名，$为自己理解为赋值后的id，格式为while； do； done
$ find ./ -name “_“ | while read id; do mv $id ${id/_/-}; done
